Приложение для курьеров MegaMall

Добро пожаловать в команду!

Рады видеть тебя в команде MegaMall! Это приложение поможет получать заказы, обновлять статусы доставки и сдавать наличные — всё в одном месте. Установи его на телефон, чтобы начать работу.

СКАЧАТЬ ДЛЯ Android
Только для Android · APK-файл

Как установить

1

Нажми кнопку «Скачать для Android» выше и дождись загрузки файла.

2

Если телефон спросит — разреши установку из неизвестных источников для браузера.

3

Открой скачанный файл и нажми «Установить».

4

Запусти приложение и войди под своим логином курьера.
